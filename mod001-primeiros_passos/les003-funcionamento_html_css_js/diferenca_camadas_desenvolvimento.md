# Camadas de Desenvolvimento

## Front-end (Client-side)

Essa camada é o que chamamos de **client-side**. O foco está na ***experiência do usuário, na parte visual e na interatividade com a qual as pessoas interagem diretamente***.

**Tecnologias:** É aqui que ficam a estrutura HTML, as regras CSS e os comandos JavaScript.


## Back-end (Server-side)

Essa camada é chamada de **server-side**. A "infraestrutura oculta" cuida da ***lógica de negócios, do processamento, da segurança e da ponte com o banco de dados***.

**Tecnologias:** Utiliza linguagens que o servidor entende, como PHP, Python, Java ou C#.


## Full Stack (Ambos)

Refere-se à *capacidade de lidar com todas as camadas* de uma aplicação. **Combina as habilidades de Front-end e Back-end** para construir e manter uma solução web completa.
