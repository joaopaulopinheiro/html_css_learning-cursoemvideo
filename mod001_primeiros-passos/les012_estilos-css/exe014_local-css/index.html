Estilo Interno (Na mesma página)

Os estilos internos são definidos na seção <head> do documento HTML usando a tag <style>. Eles se aplicam a toda a página e são mais organizados do que os estilos inline.

Exemplo de estilo interno:

Este parágrafo tem um estilo interno aplicado, com cor azul e tamanho de fonte de 18 pixels.

Este parágrafo possui essa mesma formatação, através do elemento style.

Sua principal desvantagem é que ocupa muito espaço no código HTML.
Além disso, o estilo inline possui maior especificidade do que o local, por isso este parágrafo não é azul.
