Estilo Inline (Na mesma linha)

Em um código HTML, os estilos inline são aplicados diretamente aos elementos. Para isso, basta adicionar o atributo style a qualquer elemento e definir os estilos desejados dentro de aspas.

Exemplo de estilo inline:

Este parágrafo tem um estilo inline aplicado, com cor azul e tamanho de fonte de 18 pixels.

Exemplo de outros valores do atributo style

  • Background-color - Fundo Aqua
  • Color - Texto Vermelho
  • Font-size - Tamanho da Fonte 18px
  • Text-align - Alinhamento do Texto na Direita
  • Font-family - Família da Fonte Arial

Sua principal desvantagem é que cada estilo só se aplica a um elemento por vez.
