Principais formatações em HTML5

Negrito / Destaque

Em HTML5, a tag <strong> é preferida para indicar destaque, pois é semântica. Ao contrário da tag <b> (não semântica).

Itálico / Ênfase

A tag <em> é preferida para indicar ênfase, pois é semântica. Ao contrário da tag <i> (não semântica) que representa texto em tom alternativo, como termos técnicos.

Texto marcado

A tag <mark> é usada para marcar texto com relevância, como na frase: "A Guerra Fria foi uma batalha tecnológica entre os EUA e a URSS."

Texto grande e pequeno

Em HTML4 a tag <big> era usada para criar textos grandes, antes de ser tornar obsoleta.

Em HTML5, criamos um texto pequeno usando a tag <small>

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, usando a tag <del>

Texto inserido

Usamos a tag <ins> para marcar um texto como inserido, pois é semântica. Ao contrário da tag <u> (não semântica) que representa anotação não textual, como erros de ortografia.

Texto sobrescrito e subscrito

Por fim, usamos a tag <sup>

sup> para texto sobrescrito e <sub> para texto subscrito.
