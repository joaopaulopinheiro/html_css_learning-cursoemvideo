# Tag BR

Tag que cria uma quebra de linha.

# Caracteres Especiais

Caracteres como setas, símbolos entre outros. São adicionadas no HTML com o símbolo &. (E comercial) 

**Para símbolos:** 

> & + o código do caractere. Ex: &reg; ou &lt;

# Emojis

São adicionadas no HTML com os símbolos &# + a letra X.

> &#x + o código do caractere. Ex: &#x1F596; ou &#x1F913;
