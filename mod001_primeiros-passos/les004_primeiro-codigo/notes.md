# Tags Iniciais HTML

## !Doctype

Evidencia qual a versão utilizada o HTML. Ex: ```!docytype html``` == HTML5.

## Html

Todas as marcações html da página ficarão dentro desta tag.

## Head

Configurações do site.

## Meta CHARSET

Conjunto de caracteres. Ex: ```meta charset="UTF-8"``` == permite caracteres com acentuação.

## Meta VIEWPORT

Determina que o conteúdo ocupe 100% da largura dada pelo navegador.

## Title

Título do site.

## Body

Conteúdo do site.

## Heading (H1-H6)

Tag de título para conteúdos da página.

## Hr

Tag para mostrar uma linha horizontal.

## P

Tag para criar um parágrafo.
