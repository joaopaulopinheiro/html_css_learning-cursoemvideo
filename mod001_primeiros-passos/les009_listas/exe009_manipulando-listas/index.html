Trabalhando com Listas

Podemos adicionar uma lista ordenada para representar passos ou etapas onde a ordem é importante, como no exemplo abaixo:

  1. Acordar
  2. Beber água
  3. Praticar exercício
  4. Higiene Pessoal
  5. Tomar café
  6. Definir 3 prioridades do dia
  7. Almoçar
  8. Estudar programação
  9. Reter o conhecimento
  10. Praticar gratidão
  11. Jantar
  12. Dormir

Podemos adicionar uma lista não ordenada para representar itens onde a ordem não é importante, como no exemplo abaixo:

  • Pão
  • Leite
  • Tomate
  • Alface
  • Manteiga
  • Arroz
  • Feijão

Também, podemos mesclar as listas uma dentro da outra, como no exemplo abaixo:

Meus livros favoritos

  1. Desenvolvimento Pessoal
    • Essencialismo
    • Poder do Subconsciente
    • Milagra do Amanhã

Em lista de definições, a ordem não é, necessariamente, importante e são usadas para definir termos ou conceitos.

HTML
Linguagem de marcação para a criação do conteúdo de um site, como títulos, textos, formatações, etc.
CSS
Linguagem de marcação para a criação do design de um site, como tamanho de fonte, posicionamento, cores, etc.
JavaScript
Linguagem de programação para a criação de interatividade de um site, como validações, variáveis, popups, etc.
